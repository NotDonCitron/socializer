# Tech Stack

## Core Framework
- **Astro**: Static site generator and main web framework (v5.x).
- **React**: UI library for interactive components (via `@astrojs/react`).

## Styling & Assets
- **CSS**: Standard CSS (likely scoped or global).
- **Fontkit/SVGO**: Asset optimization tools (previously patched, now standard).

## Tools & Workflow
- **Vibe Kanban**: Task management (standalone mode via `npx vibe-kanban`).
- **NPM**: Package manager.
- **Git**: Version control.

## Deployment
- **GitHub Pages**: Hosting target (static export).
