# Product Context: AI Agent Radar / Socializer Site

## Vision
A central hub for tracking and visualizing the impact of AI Agents and social automation tools.

## Core Features
- **Updates Feed**: A list of weekly updates and news about AI agents (e.g., LangChain, MCP).
- **Task Management**: Integrated (but loosely coupled) Kanban board for managing development.
- **Content-Driven**: Markdown-based content collections for easy updates.

## Current State
- **Status**: Development/Prototyping.
- **Recent Fixes**: Resolved build stability issues by isolating the Kanban tool.
- **Next Goals**: Expand content with more GitHub project showcases.
